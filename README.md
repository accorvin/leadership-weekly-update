# sherard-update

A Claude Code skill for collaboratively writing weekly status updates to Confluence for Sherard and peers on the AI Engineering leadership team.

Heavily inspired by [etirelli/weekly-update](https://github.com/etirelli/weekly-update).

## What it does

- Guides you through adding updates interactively
- Enforces a high bar — pushes back on low-value items and items that don't fit the expected categories
- Writes a polished, executive-appropriate entry to a shared Confluence page
- Creates the weekly page automatically if it doesn't exist

## Setup

Run the skill once with no arguments. It will walk you through first-time configuration.

## Usage

```
/sherard-update <description of what you want to report>
```

Or run with no arguments for interactive/conversational mode.
